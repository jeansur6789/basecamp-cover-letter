Basecamp Classic does not offer the feature of automated recurring events at this time. However, you can add it manually each time you want it to occur.

For recurring tasks, Basecamp does not have this feature implemented, but there is downloadable software that can easily be integrated with your Basecamp 3 account: "To-do Helpers" is a great app that will help automate recurring to-do lists so you won't have to do it manually. "Recur" is another tool for Basecamp 3 that will help manage your to-do lists.
